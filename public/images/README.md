## Bildoptimierung

Für optimale Ladezeiten:

1. **Format:** JPEG für Fotos (WebP für moderne Browser als Alternative)
2. **Kompression:** 75-85% Qualität
3. **Dateigröße:** Maximal 300 KB pro Bild
4. **Tools:** TinyPNG, ImageOptim, oder Squoosh
